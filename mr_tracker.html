Match Rating Tracker

    General Operations

  • This tool is a tracking tool for tracking the trend of your rank in Marvel Rivals.

  • Submit a rating change after each game using the input form. The chart automatically updates and adjusts your rank color and label based on your total rating.

  • You can export your current chart data to a JSON file to save your progress.

  • You can also import previously saved JSON files to restore your chart data and continue tracking.

    • Import File Specifications

  • The import file must be a JSON file.

  • The JSON structure must include:
    • labels: an array of game labels (e.g., "Game 1", "Game 2", etc.).
    • dataset: an array of corresponding rating values (numbers) matching the labels.

  • Example format: 
            {
          "labels": ["Game 1", "Game 2"],
          "dataset": [150, 300]
        }

User Guide

Welcome to the Overwatch Tracker! This tool allows players to dynamically view their gameplay statistics through a series of automatically generated bar graphs. By entering your BattleTag, you can instantly retrieve and visualize your current performance data, helping you better understand your strengths and areas for improvement.

    General Operations

  • Entered battletags must replace the "#" symbol with the "-" symbol. Your profile must also be set to public status.

  • Use the selection feilds below to select which hero and statistic you want to see vizualized.

  • Clicking the "add player Data" button will add the slected data to the "Custom stat View", any data can be mixed and matched here.

  • Just by using the selection boxes, the other two graphs will autopopulate to show the requested hero across stats or a specific stat for each hero.

    • Manual Mode

  • To enable manual mode check the box under the username that says "Toggle Manual Input Mode"
  • If collection of data is impossible this page will open in manual input mode. In this mode you can manually construct the data yourself with an intutive GUI
  • Pressing the "add stat" button in manual mode will select whatever player you currently have inputted into the player text box and attempt to read that players stats
  • In Manual Mode the selection boxes located at the top of the GUI work the same as normal mode. However, the data within them is gotten by manually entering the data in the GUI below and pressing the add stat button.

  • A Final note for manual mode is that it is mainly intended for when we are unable to access any player data, so switching between the two modes often is not recommended as it could lead to abnormal behavior

Resources

This project provides tools and guides for Overwatch 2 and Marvel Rivals players to improve their gameplay. Whether you're a beginner or a seasoned player, you'll find valuable insights and tools to enhance your skills and strategies.

Data Sources

  • Overfast API - A comprehensive API for Overwatch 2 data, including hero stats, maps, and game modes.
  • Official Overwatch Heroes - Detailed information about Overwatch heroes, including abilities, lore, and gameplay tips.
  • Marvel Rivals Offical Website - Has all sorts of information about the game and different details.

Topic

This project focuses on providing strategies, guides, and tracking tools for competitive gaming. Our goal is to help players analyze their performance, learn from their mistakes, and achieve their full potential in Overwatch 2 and Marvel Rivals.

Future Plans

We aim to expand the project by adding more features, such as personalized performance analysis and more in-depth guides. Additionally, we plan to integrate more games and provide a platform for players to learn new strategies and improve their experiences.
